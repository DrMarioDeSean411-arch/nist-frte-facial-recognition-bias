Compounding harm: the closed feedback loop

Racially biased policing does not merely predate algorithmic surveillance — it actively produces the conditions that make algorithmic surveillance more harmful. Click any node to see the supporting statistic and source.

What this shows: Each node in this loop is documented independently in the literature. Together they form a self-reinforcing system: disparate policing produces skewed data, skewed data produces biased algorithms, biased algorithms produce more false leads in over-policed communities, false leads produce more arrests, more arrests produce more skewed data. The NIST FMR disparities documented in visualizations 1–4 are one node in this loop — not a self-contained technical failure.

Methodological note: Causal arrows represent relationships documented across multiple peer-reviewed sources. This is not an experimentally established direct causal chain; it is a structural argument grounded in documented co-occurrence and mechanism. See footer citations.
